# HTML Pages - Reminder System

This folder contains standalone HTML versions of all pages from the Reminder System.

## Pages Included

1. **navigation.html** - Quick navigation to all pages (START HERE!)
2. **index.html** - Login page
3. **dashboard.html** - Dashboard with statistics
4. **employees.html** - Employee management
5. **bulk-upload.html** - Bulk upload functionality
6. **events.html** - Event configuration
7. **templates.html** - Template editor
8. **reports.html** - Reports and analytics
9. **settings.html** - System settings

## Features

- ✅ Fully responsive design (mobile, tablet, desktop)
- ✅ Mobile drawer menu
- ✅ Tailwind CSS styling
- ✅ All pages accessible without login
- ✅ All pages linked together via sidebar
- ✅ Demo functionality

## Usage

### Option 1: Quick Navigation
1. Open `navigation.html` in a web browser
2. Click any card to navigate to that page
3. All pages are directly accessible

### Option 2: Direct Access
1. Open any HTML file directly (e.g., `dashboard.html`)
2. Navigate using the sidebar menu
3. No login required - all pages are accessible

### Option 3: Login Flow
1. Open `index.html` in a web browser
2. Login with any email/password (demo mode)
3. Navigate through all pages using the sidebar menu

## Notes

- **All pages are now accessible without login** - authentication checks removed
- These are static HTML pages with demo functionality
- Forms show alerts instead of actual API calls
- All pages are responsive and mobile-friendly
- Use `navigation.html` for quick access to all pages
